# template-config4

A template for starting a new config4 quickly.

## Install

Install the [`template`](https://github.com/fabiospampinato/template) CLI tool in your system.

Install the template locally with:

```sh
template install frog-o/template-config4 config4
```

## Usage

Create a new config4 with:

```sh
template create config4 my-config4
```
